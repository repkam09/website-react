Lifeforce Information

Lifeforce is the codename for the set of REST apis that powers this website, several android apps, other webapps, my Minecraft Server and more. All of this code is open source and can be found on my GitHub account here

Some of these APIs will require authentication, while others are totally public.


Lifeforce can also be configured to respond to GitHub webhooks to automatically redeploy any project that is configured with a path to the source location and given an update script to run.
